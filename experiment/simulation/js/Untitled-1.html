Half wave/Full wave Rectifier

Objective:

  1.  To differentiate between input and output waveforms in Half wave/full wave rectifier
  2.  To calculate the Ripple Factor

Apparatus Used:

Diodes,Step down Transformer,Resistor,DSO

DSO

Diode

Resistor

Transformer




N1:primary turns

N2:secondary turns

Resistor: Ω

Frequency: 50Hz





Instructions

  1. Select the stepdown transformer from the menu and change the number of turns in the primary and secondary
  2. Select the resistance value, connect input signals to channel 1 (C1) of the DSO.
  3. Analyze the input waveform without a diode
  4. Connect two diodes in parallel with the resistance (RL) and take the Output across RL
  5. Connect the Output to channel 2 (C2) of the DSO.
  6. Measure V(rms) , Vdc , Vac From the DSO.
  7. Calculate the Ripple factor of the Full wave rectifier using the given formula Î³=sqrt((Vrms/Vdc)^2-1)
